**Lothian Birth Cohort 1936 Association Study of Clinical Phenotypes with WMH Volume Analysis Repository** 
---

This folder contains scripts and supporting documentation for analyses conducted as part of a LBC1936 study, which explored associations between clinical phenotypes and white matter hyperintensity volume.

#### Project Overview

The primary objective of this analysis was to examine associations of clinical phenotypes with white matter hyperintensity volumes in a subset of participants from the LBC1936 using a SEM analysis framework.

#### Contents

- Data processing scripts
  
- Statistical modelling scripts (e.g., structural equation modelling)

Analyses were conducted in R (several versions, as required).

Key packages include:

- haven
- lavaan
- psych

#### Data Availability

LBC1936 data are not included in this repository due to data sharing restrictions. Scripts assume access to the LBC1936 dataset.

Variable names reflect the original (“native”) naming conventions used in the source datasets. Users adapting these scripts to other datasets will need to modify variable names accordingly.
