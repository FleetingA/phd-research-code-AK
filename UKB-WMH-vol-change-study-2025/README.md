**UK Biobank Association Study of Baseline Phenotypes with WMH Volume Change Analysis Repository** 
---

This folder contains scripts and supporting documentation for analyses conducted as part of a UKB study, which explored associations between baseline phenotypes and white matter hyperintensity volume change.

#### Project Overview

The primary objective of this analysis was to examine associations of baseline phenotypes with white matter hyperintensity volume change over a median of 2.3 years in 4329 participants from the UK Biobank. 

#### Contents

- Data processing scripts
  
- Data analysis scripts

Analyses were conducted in R (several versions, as required).

Key packages include:

- dplyr
- psych
- ggplot2

#### Data Availability

UK Biobank data are not included in this repository due to data sharing restrictions. Scripts assume access to the UK Biobank dataset.

Variable names reflect the original (“native”) naming conventions used in the source datasets. Users adapting these scripts to other datasets will need to modify variable names accordingly.
